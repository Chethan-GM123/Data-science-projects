# Heart Disease Prediction

## Project Overview
This project predicts whether a patient is likely to have heart disease using machine learning techniques.

## Problem Statement
Heart disease is one of the leading causes of death. This project aims to help in early detection using data analysis.

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / Seaborn

##  Process
- Data Collection
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Model Building (Classification)
- Model Evaluation

## 📈 Result
The model achieved good accuracy in predicting heart disease and helps in identifying risk factors.
