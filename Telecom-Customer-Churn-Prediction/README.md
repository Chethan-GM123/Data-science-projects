# Telecom Customer Churn Prediction

## Project Overview
- Machine learning project developed during my Data Science internship at Rubixe Technologies Pvt. Ltd., Bangalore
- Developed a machine learning model to predict customer churn and identify customers likely to leave 

## Objectives
- Identify customers at risk of churn.
- Help telecom companies improve customer retention strategies.

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Machine Learning Models
- Logistic Regression
- Random Forest

## Results
- Achieved approximately 85% accuracy in predicting customer churn.

## Business Impact
The model provides actionable insights that help telecom companies take proactive measures to retain customers.
