# 🏦 Portuguese Bank Term Deposit Prediction

## Project Overview

This project predicts whether a customer will subscribe to a term deposit based on banking data.

## 🎯 Objective

To analyze customer data and build a machine learning model for predicting term deposit subscription.

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib / Seaborn

## 📊 Dataset

* Portuguese Bank Marketing Dataset
* Features include age, job, marital status, balance, etc.

## 🚀 Steps Performed

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Building
5. Model Evaluation

## 📈 Model Performance

* Accuracy: 85% (update if needed)

## Files in this Project

* `Portuguese Bank Term Deposit Prediction chethangm.ipynb` → Main notebook

##  Conclusion

The model helps banks identify potential customers for term deposits, improving marketing strategies.
